Line break standards between CSV and HTML

In case you were wondering

1/19/2024

In CSV files (when editing the file as text), you can indicate a spreadsheet line break between rows using either a comma and line break or just simply a line break.
Method 1:
This is cell 1 on row 1,This is cell 2 on row 1,
This is cell 3 on row 2,


Method 2:
This is cell 1 on row 1,This is cell 2 on row 1
This is cell 3 on row 2

In HTML files, document line breaks are indicated by the "<br>" tag. This tag can be used anywhere in the document (except within a few other tags, such as the "<script>" tag).

In summary, an HTML document's line breaks are achieved using the "<br>" tag while line breaks for a CSV spreadsheet are physically inserted with the 'enter' key. These methods can't really be intermixed. Until next time!
