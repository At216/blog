This project is the literal flagship of my subdomain. I originally registered for this subdomain solely to host it. This website generates a NodeJS server for download in you browser based on provided file path parameters. I use this site often, when I want to run a local HTTP server on a computer I haven't written a server for. With a server from this site, you can serve HTML, CSS, JavaScript, PNG, JPEG, and GIF files. It should be somewhat straightforward to edit to increase compatibility for other filetypes as well (assuming you have some knowledge of NodeJS or JavaScript). Visit this project at ServersNJS.w3spaces.com/.

This is one of my favorite (and most wanted) projects. Ever since I began programming in HTML, I wanted an simple and online HTML code editor and renderer. I tried to build this as a budding programmer, but eventually encountered difficulties. Just recently (about 7 years later), I was experimenting with AJAX and had a revelation moment about rendering HTML code in the browser. I sat down and tinkered with textarea and iframe elements, ultimately constructing the project I had wished for for so long. Visit this project at ServersNJS.w3spaces.com/HTML-editor.html.
